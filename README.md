# vocasnap-support
                                                                                                                   
                                                                                                                                              
  よくある質問    

  Q. 1日に使える回数は？                                                                                                                             
  A. 1日16回まで無料でご利用いただけます。回数は毎日0時にリセットされます。
                                                                                                                                                     
  Q. 音が出ない   
  A. iOSのマナーモード（着信音スイッチ）をオフにしてください。                                                                                       
     スイッチをオンにすると読み上げ音声が再生されます。                                                                                              
                                                                                                                                                     
  Q. カメラが起動しない                                                                                                                              
  A. 設定 > Vocasnap > カメラのアクセスを「オン」にしてください。                                                                                    
                                                                                                                                                     
  Q. 認識精度を上げるには？
  A. 単語が映った部分をなるべく大きく・明るくフレームに収めると                                                                                      
     精度が上がります。                                                                                                                              
                                                                                                                                                     
  Q. 画像はどこかに保存されますか？                                                                                                                  
  A. 画像はAI分析のためにのみ送信され、当方のサーバーには保存されません。                                                                            
     詳細はプライバシーポリシーをご参照ください。

  📷  撮るだけで単語を認識

      難しい操作は一切不要。                                                                                                                         
                                                                                                                                                     
  🔊  ネイティブ発音で読み上げ                                                                                                                       
      ボタン1つで正しい発音を確認。                                                                                                                  
                                                                                                                                                     
  📖  例文・品詞まで表示                                                                                                                             
      意味だけでなく、使い方も一緒に学べる。                                                                                                         
                                                                                                                                                     
  💾  画像をそのまま保存・共有                                                                                                                       
      学習メモとして端末に保存したり                                                                                                                 
      友人と共有することも可能。
